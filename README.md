# 機能

- 非常に普通な「投稿サイト」
- 一日で消える仕様

# デプロイ

- [https://my-uncle-blog-210322.herokuapp.com/](https://my-uncle-blog-210322.herokuapp.com/)


## ユーザーのメリット
- 事前に文章をまとめてからツイートすることで、不安定な文章をツイッターに投稿しなくなる
- 1日で投稿が消えるので、気軽に文章を作成できる
- データがネットに蓄積されないので、世の中からヘイトツイートも減る
- 個人情報が流出することがない

## 運営サイドのメリット
- データの管理が楽
