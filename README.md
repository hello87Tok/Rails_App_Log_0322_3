# 作成経緯

## 課題
- 世の中にヘイトツイートが多すぎる
- ツイートの誤字を修正することができない
- 個人情報がずっと残っているという漠然とした不安

## ユーザーのメリット
- 事前に文章をまとめてからツイートすることで、不安定な文章をツイッターに投稿しなくなる
- 1日で投稿が消えるので、気軽に文章を作成できる
- データがネットに蓄積されないので、世の中からヘイトツイートも減る(バックアップはその場でTwitter共有のみ)
- 個人情報が流出することがない

## 運営サイドのメリット
- データの管理が楽
